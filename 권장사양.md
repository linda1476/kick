## 🎯 사용 시나리오별 권장

### 학습 및 테스트용

**최소 사양으로도 충분:**
- CPU: i5 이상
- RAM: 8GB
- 웹캠: 720p
- **실행**: `python webcam.py --device cpu`

### 데모 및 프레젠테이션

**AMD GPU 권장:**
- GPU: Radeon RX 5000 시리즈 이상
- RAM: 16GB
- 웹캠: 1080p
- **실행**: `python webcam_amd.py --device dml`

### 실시간 프로덕션

**NVIDIA GPU 필수:**
- GPU: RTX 3060 이상
- RAM: 16GB
- 웹캠: 1080p
- **실행**: `python webcam_tracking.py --device cuda`

## 📦 소프트웨어 요구사항

### 필수 패키지

```
torch >= 2.0.0
torchvision >= 0.15.0
opencv-python >= 4.8.0
numpy >= 1.24.0
pillow >= 10.0.0
```

### GPU별 추가 패키지

**AMD GPU (Windows):**
```bash
pip install torch-directml
```

**NVIDIA GPU:**
```bash
# CUDA 11.8
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

# CUDA 12.1
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
```

## 🔍 성능 최적화 팁

### CPU 모드 최적화

```bash
# 신뢰도를 높여 처리량 감소
python webcam.py --device cpu --conf 0.7

# 해상도 낮추기 (코드 수정 필요)
cap.set(cv2.CAP_PROP_FRAME_WIDTH, 320)
cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 240)
```

### AMD GPU 최적화

```bash
# 적절한 신뢰도 설정
python webcam_tracking.py --device dml --conf 0.6

# DirectML 최신 버전 사용
pip install --upgrade torch-directml
```

### NVIDIA GPU 최적화

```bash
# CUDA 최신 드라이버 확인
nvidia-smi

# 최신 PyTorch 사용
pip install --upgrade torch torchvision
```

## 📝 참고사항

- **노트북**: 일반적으로 데스크톱보다 10-20% 성능 저하
- **배터리 모드**: GPU 성능이 30-50% 감소할 수 있음
- **멀티태스킹**: 다른 프로그램 실행 시 FPS 감소
- **웹캠 품질**: 1080p 웹캠 권장, 4K는 불필요
