# 동적 객체 추적 실행 가이드

## ✨ 새로운 기능

동적 객체 추적이 추가되었습니다!

### 추가된 기능

- ✅ **객체 ID 부여**: 각 객체에 고유 ID 할당
- ✅ **이동 경로 표시**: 객체가 움직인 궤적을 선으로 표시
- ✅ **속도 및 방향**: 화살표로 이동 방향과 속도 표시
- ✅ **지능형 추적**: 프레임 간 같은 객체를 자동 인식

## 🚀 실행 명령어

### AMD GPU 사용

```bash
python webcam_tracking.py --device dml
```

### NVIDIA GPU 사용

```bash
python webcam_tracking.py --device cuda
```

### CPU 사용

```bash
python webcam_tracking.py --device cpu
```

## 📊 화면 표시 정보

각 객체에 다음 정보가 표시됩니다:

```
ID3 Car 0.87 5.2px/f
  ↑   ↑   ↑     ↑
  ID  클래스 신뢰도 속도
```

## 🎨 시각 효과

1. **바운딩 박스**: Tesla FSD 스타일 색상별 박스
2. **이동 경로**: 객체가 지나간 경로를 선으로 표시
3. **방향 화살표**: 현재 이동 방향과 속도를 화살표로 표시
4. **ID 레이블**: 각 객체의 고유 ID 표시

## ⚙️ 조작법

| 키 | 기능 |
|---|---|
| `q` | 종료 |
| `s` | 스크린샷 저장 |
| `+` | 신뢰도 증가 |
| `-` | 신뢰도 감소 |

## 📝 전체 옵션

```bash
# 기본 실행 (AMD GPU)
python webcam_tracking.py --device dml

# 신뢰도 조정
python webcam_tracking.py --device dml --conf 0.6

# 다른 카메라
python webcam_tracking.py --device dml --camera 1

# 모든 옵션
python webcam_tracking.py --device dml --conf 0.5 --camera 0
```

## 🔍 비교

| 스크립트 | 기능 | 용도 |
|---------|------|------|
| `webcam.py` | 기본 감지 | 단순 객체 감지 |
| `webcam_amd.py` | AMD GPU 지원 | AMD GPU 가속 |
| **`webcam_tracking.py`** | **동적 추적** | **객체 추적 + 경로 + 속도** |

## 💡 사용 예시

### 예시 1: 보행자 추적

웹캠 앞에서 걸으면:
- `ID0 Person 0.92 3.5px/f` 레이블 표시
- 초록색 이동 경로 선
- 이동 방향 화살표

### 예시 2: 차량 추적

차량이 지나가면:
- `ID1 Car 0.85 8.2px/f` 레이블 표시
- 파란색 이동 경로 선
- 속도에 비례한 긴 화살표
